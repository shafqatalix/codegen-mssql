# codegen-mssql

## Setup Development Environment

- pip install -r dev.txt
- pip install -r requirements.txt

## How to use CLI:

codegen-mssql parms:

| Short | Long Name          | Description                   |
| ----- | ------------------ | ----------------------------- |
| -c    | --connectionstring | Database connection string    |
| -t    | --target           | Target output , default is C# |

## Supported Targets

- [x] C# (default)
- [ ] Typescript
- [ ] Java
- [ ] Rust
